# Olostep Projects

A collection of projects built with Olostep for web scraping, crawling, search, and AI-powered data extraction.

## AI Agents & Assistants

### Real-Time Voice Agent with LiveKit, Gemini & Olostep
Build a real-time voice agent that listens and responds using Google Gemini's Live API with live web search via Olostep.

- **Guide**: [How to Build a Real-Time Voice Agent](https://www.olostep.com/blog/how-to-build-real-time-voice-agent)
- **GitHub**: [Studio1HQ/livekit-web-search-agent](https://github.com/Studio1HQ/livekit-web-search-agent)
- **Stack**: Python, LiveKit, Gemini Live API, Olostep SDK

### AI Searchable Knowledge Base with Olostep & Weaviate
Crawl websites with Olostep, store data in Weaviate, and build an AI chat app that answers with cited sources.

- **Guide**: [How to Build AI Searchable Knowledge Base](https://www.olostep.com/blog/build-ai-searchable-knowledge-base-with-olostep-weaviate)
- **GitHub**: [Studio1HQ/website-knowledge-base](https://github.com/Studio1HQ/website-knowledge-base)
- **Stack**: Python, Streamlit, Weaviate, Agno, Olostep SDK

### Agentic Market Research & Trend Analysis
End-to-end multi-agentic trend analysis system that pulls credible sources, extracts real signals, and generates clean markdown briefs.

- **Guide**: [Agentic Market Research & Trend Analysis with Olostep](https://www.olostep.com/blog/agentic-market-research-olostep)
- **GitHub**: [kingabzpro/agentic-market-research-olostep](https://github.com/kingabzpro/agentic-market-research-olostep)
- **Stack**: Python, OpenAI Agents SDK, GPT-5.2, Olostep API, Gradio

### Blog Writing Agent with Olostep
A reliable blog writing agent using Olostep as the web access layer for research, extraction, and source-grounded drafting.

- **Guide**: [How to Build a Reliable Blog Writing Agent](https://www.olostep.com/blog/blog-writing-agent)
- **GitHub**: [olostep-api/ai-content-engine](https://github.com/olostep-api/ai-content-engine)
- **Stack**: Python, Olostep SDK

### Multi-Agent Research Assistant
Multi-agent research assistant using OpenAI Agents SDK and Olostep to produce sourced Markdown research reports with Manager, Judge, and Analyst agents.

- **GitHub**: [kingabzpro/Multi-Agent-Research-Assistant](https://github.com/kingabzpro/Multi-Agent-Research-Assistant)
- **Demo**: [HuggingFace Space](https://huggingface.co/spaces/kingabzpro/Multi-Agent-Research-Assistant)
- **Stack**: Python, OpenAI Agents SDK, Olostep, Reflex, ReportLab
